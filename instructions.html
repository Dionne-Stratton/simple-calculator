Scaffolding Out a New Project

Follow these instructions to scaffold out a simple project in an html file with an empty style tag and script tag.

  • Create a folder for your project
  • Open the folder in VS Code
  • Create a new html file and name it index.html
  • Copy the code below and paste it into your index.html file
  • Save the file
  • Open the file in your browser using live server

Code

            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <title>New Project</title>
                    <style>
                        /* Add your styles here */
                    </style>
                </head>
                <body>
                    <script>
                        // Add your scripts here
                    </script>
                </body>
                </html>

That's it! You now have a simple html file with an empty style tag and script tag. You can now start adding your own styles and scripts to build out your project.

For more information on scaffolding out a new project, visit the W3Schools HTML Tutorial.

Adding Functionality

To add functionality to your project, you can use JavaScript to create interactive elements and dynamic content. You can add event listeners to respond to user actions and manipulate the DOM to update the content, you will learn this in depth in sprint 5. Below is sample code of accessing the DOM and adding an event listener to a button element. You will need to have a button element with the class name "buttons" on it's parent element and an element (such as a div) with the class name "display" in your HTML file. 

            
                // Get the button element
                const display = document.querySelector(".display");
                const buttons = document.querySelectorAll(".buttons button");

                // Add an event listener to the button
                buttons.forEach((button) => {
                    button.addEventListener("click", () => {
                        do your logic here
                    });
                });
